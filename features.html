Features

Operating System

The operating system in Erlang-embedded is based on the Ångström Linux distribution.

Included software

  • The olsr adhoc wireless mesh routing daemon
  • Erlang R13B04

Erlang

The Erlang release that is included with the distribution is cross compiled for the ARM architecture version 7-A.

The distribution contains the Erlang runtime system and the libraries stdlib and kernel. In order to leave more room for application data, the Erlang emulator has been built with size optimizations and the included Erlang libraries are compressed. The executables and beam files have also been stripped.
